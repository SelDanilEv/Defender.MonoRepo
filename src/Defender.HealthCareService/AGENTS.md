# AGENTS Guide: Defender.HealthCareService

## Purpose
- Backend API for health care tracking: temperature, medication intake, and sleep intervals.

## Project entry points
- Solution: `Defender.HealthCareService.sln`
- Startup project: `src/WebApi/WebApi.csproj`
- App entry: `src/WebApi/Program.cs`

## Commands
- Run: `dotnet run --project src/WebApi/WebApi.csproj`
- Build: `dotnet build Defender.HealthCareService.sln`
- Test: `dotnet test Defender.HealthCareService.sln`
